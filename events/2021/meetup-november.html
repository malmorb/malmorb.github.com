Bruno Sutic: Async Ruby

Async is an interesting new concurrency paradigm in Ruby, best described as threads with none of the downsides. It's powerful, scalable, and integrates seamlessly with the existing Ruby code.

This talk will explain why Matz invited 'async' gem to Ruby's standard library and why Ruby added so many new language features, just to work better with Async.

Async Ruby was in stealth development mode for years, and now it's finally ready for prime-time. Come learn about the most innovative feature added to Ruby in years.
